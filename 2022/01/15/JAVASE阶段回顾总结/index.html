2 基础语法

  • 注释:
    • 行内注释://
    • 多行注释:/*…*/
    • 文档注释:/** */
      • javadoc生成帮助文档
  • java关键字:
    • abstract:抽象属性
      • 抽象类可以扩展增加子类,但不能实例化
        • 当抽象类的子类实现所有的抽象方法时,可以实例化抽象类的子类(具体类)
      • 抽象方法不在声明它的类中实现,但必须在某个子类中重写
      • 采用抽象方法的类是抽象类,该类必须声明为abstract
      • 抽象属性不能使用static,private或final方法
        • 这些方法不能被重写,不能在子类中实现,且final类不能有子类
    • assert:断言,程序调试使用
    • boolean:基本数据类型,布尔量
    • break:提前跳出一个块
    • byte:基本数据类型,字节类型
    • case:在switch语句中的一个分支
    • catch:异常处理时,捕获异常
    • char:基本数据类型,字符类型
    • class:声明一个类
    • const:保留关键字
    • continue:回到一个块的开始处
    • default:在switch语句中的默认分支;Java8中的接口函数的默认实现
    • do:在do-while结构中
    • double:基本数据类型.双精度浮点数
    • else:条件语句,条件不成立的分支
    • enum:枚举类型
    • extends:表明一个类是另一个类的子类,或另一个接口(继承关系)
      • 用在class或interface声明中,指示所声明的类或接口为子类
    • final:最终属性,表明一个类不能派生子类,或成员方法不能被覆盖,或成员域的值不能被修改
      • 一个类或方法,不能同时为final或abstract的,可扩展与不能重写的冲突
    • finally:处理异常时,肯定会被执行的语句块
      • 通常包含清理代码
    • float:基本数据类型,单精度浮点数
    • for:循环
    • if:条件语句
    • implements:表明一个类实现了给定的接口
    • import:访问指定的类或包
    • instanceof:测试一个对象是否是指定类型的实例对象
    • int:基本数据类型,整型
    • interface:接口
      • 声明新的Java接口,接口是方法的集合
      • 一个类可以实现多个接口
    • long:基本数据类型,长整数
    • native:本地方法
    • new:创建新实例对象
    • null:表示无值
    • package:包
    • private:私有模式,访问控制,类可见
    • protected:保护模式,访问控制,包可见
    • public:共用模式,访问控制
    • return:从成员方法中返回数据
    • strictfp
    • short:基本数据类型,短整数
    • static:静态属性
      • 内部类,内部方法或字段
      • 意味着在声明该实体的类的任何特定实例外部可用
      • 内部类可以被其他类实例化和引用
    • super:当前对象的父类引用,或父类构造方法
    • switch:分支语句
    • synchronized:代码同步执行
    • this:当前实例对象的引用
    • throw:抛出一个异常
    • throws:声明在当前定义的成员方法中所有要抛出的异常
    • transient:声明不用序列化的成员域
    • try:尝试一个可能抛出异常的程序块
    • void:成员方法没有返回值
    • volatile:表明两个或多个变量必须同步产生变化
    • while:循环
  • 数据类型
    • 基本数据类型:
      • 整数:byte(1字节),short(2字节),int(4字节),long(8字节)
        • 二进制0b开头,十六进制0x开头,八进制0开头
      • 浮点数:float(4字节),double(8字节)
        • BigDecimal大数据类型用于金融计算,精确计算
      • 字符:char(2字节)
        • 编码:ASCII(中文1字节),UTF-8(中文3字节),Unicode(中文2字节)
          • 转义字符:\n,\b,\r,\t,’,",\\
      • 布尔值:boolean(1位)
    • 引用数据类型:
      • 类
      • 接口
      • 数组
  • 类型转化
    • 自动类型转换:低转高
    • 强制类型转换:高转低
  • 变量和常量
    • 变量定义
    • 作用域:
      • 类变量:static变量
      • 实例变量
      • 成员变量(局部变量):方法内变量
    • 常量:final变量
    • 命名规范:
      • 驼峰命名(变量,方法)
      • 类命名(首字母大写,驼峰命名)
      • 常量(大写+下划线)
  • 运算符:
    • 算数运算符:+ - * / % ++ –
    • 赋值运算符:=
    • 关系运算符:> < >= <= == != instanceof
    • 逻辑运算符:&& || !
    • 位运算符:& | ^(异或) ~ >> << >>>
    • 条件运算符:? :
    • 扩展运算符: += -+ *= /=
  • 包机制
    • 域名倒写
    • 防止命名冲突
    • package
    • import
  • JavaDoc
    • Java帮助文档

      • @param

      • @throws
